## Next steps would be to read in various EN record types and examine if there are any changes between the crosswalk

## By creating a crosswalk between reference management software there would not be a need for CiteSource to build in crosswalks for multiple databases
## instead, users could first upload .ris to their preferred reference software before uploading it into CiteSource
 
## CiteSource could then more accurately join and select preferred medatada
